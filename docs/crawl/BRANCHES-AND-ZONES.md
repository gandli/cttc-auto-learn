## 🔗 相关 API

```
# 获取所有分院/专区列表
GET /api/v1/system/home-config/organization

# 获取指定分院的导航菜单
GET /api/v1/system/home-nav?homeConfigId={configId}

# 获取指定分院的内容模块
GET /api/v1/system/home-module?homeConfigId={configId}

# 获取指定分院的首页内容
GET /api/v1/system/home-content?moduleHomeConfigId={moduleId}

# 获取指定分院的页脚
GET /api/v1/system/home-footer?homeConfigId={configId}

# 获取公司信息
GET /api/v1/system/company-sync/get-company-by-id?id={orgId}
```

## 📝 URL 模式

### 分院/专区首页
```
https://mooc.ctt.cn/#/home?configId={configId}
```

### 分院内页面
```
https://mooc.ctt.cn/#/{navUrl}?configId={configId}
```

示例:
- 专卖分院学习资源: `https://mooc.ctt.cn/#/study/branch/index?configId=03e2ef50-92fe-4ad4-9391-6289127ab901`
- 物流分院政策法规: `https://mooc.ctt.cn/#/policy-statute/index?configId=2ed3163d-d990-4ed8-9538-9860b01aad63`
